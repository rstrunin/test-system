Данный сервис представляет возможность быстро создавать тесты для проверки знаний обучающихся.

Инструкции для обучающих и обучающихся приведены ниже:

Для обучающих

1. Для создания нового теста необходимо перейти на главную страницу сервиса. Далее, нажать кнопку "Добавить тест". В появившейся вкладке необходимо вести название тестирования, который будет использоваться для поиска в списке тестов учениками. Название должно быть уникально.


2. После заполнения названия, необходимо добавить нужные вопросы и ответы к каждому вопросу. Вопросов и ответов может быть произвольное число.


3. Необходимо заполнить комментарии к результатам выполнения, в зависимости от набранных баллов.


4. Выбрать тип отслеживания статистики. Открытая статистика доступна всем пользователям сайта, закрытая статистика доступна по кодовому слову.

Для обучающихся

1. Для прохождения теста необходимо выбрать тест по названию на главной странице сайта.


2. Во всплывающем окне ввести свой идентификатор пользователя (имя, фамилия, логин).


3. Выполнить тестирование.
